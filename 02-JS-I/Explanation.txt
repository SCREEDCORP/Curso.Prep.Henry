funciones :

Las funciones  son un conjunto de instrucciones de codigo  que calculan  un valor 
o realizan alguna tarea   donde cuentas con un valor de entrada y una de salida .
imagina que tienes una  llave  que puede abrir todas las puertas de tu hogar , 
las funciones son similares una vez las declares las puedes usar las veces que quieras
dentro de tu codigo.  
existen 3 formas para construir una funcion, la anatomia de una funcion comienza
con la palabra clave function luego le asignas el nombre de la funcion  de 
preferencia debe  describir lo que hace tu funcion luego entre parentesis 
para que puedas asignarle  los parametros a tu funcion  y luego dentro de 
corchetes le defines las instrucciones. Una vez declarada tu funcion, podemos usar 
variables dentro de tu funcion, recuerda que puedes agregar mas de un argumento solo debes 
agregar comas entre ellos.
